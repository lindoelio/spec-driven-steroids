# general — General Purpose Review

**Posture:** Balanced

## Primary Focus

- All dimensions evaluated equally
- No specific posture bias
- Covers breadth without deep specialization

## Secondary Focus

- No special emphasis on any particular aspect

## What Blocks Approval

- Critical issues that affect any dimension
- Safety violations
- Security issues

## What Is Nit

- Minor style preferences
- Documentation suggestions
- Naming improvements

## Notes

The `general` module represents a balanced review that applies all core dimensions equally without favoring any specific posture. When `general` is detected or no specific type is identified, this module provides the default review strategy.
